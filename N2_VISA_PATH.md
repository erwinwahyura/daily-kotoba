# N2 VISA Path - Engineer Visa JLPT Requirements 🇯🇵

**Target:** JLPT N2 (CEFR B2) for Japan Engineer Visa  
**Current:** Starting from N5 foundation  
**Timeline:** 14 months (realistic for working professional)

## The New Policy

Japan now requires **JLPT N2 level proficiency** (or CEFR B2 equivalent) for engineering visa applications. This is a significant increase from previous requirements.

**Source:** Mainichi Shimbun, April 2026

## Your 14-Month Journey

### Phase 1: Foundation Lock (Months 1-2)
**Goal:** 100% N5 mastery - No gaps allowed

**Daily Schedule:**
- **Morning:** 20 N5 vocab reviews (Kotoba daily word)
- **Lunch:** 5 N5 grammar pattern reviews
- **Evening:** Kanji writing practice (15 min)
- **Weekend:** Full N5 mock test

**Milestones:**
- [ ] 800 vocab words (100%)
- [ ] 100 kanji characters
- [ ] All particle basics mastered
- [ ] Can handle simple daily conversations

**Key Focus:**

- Basic conjugation (ます・て・た・ない)
- Numbers, time, dates
- Family, home, daily activities

### Phase 2: Bridge to Intermediate (Months 3-5)
**Goal:** Complete N4 + Start N3 Grammar

**Daily Schedule:**

- **Midday:** 5 N3 grammar patterns
- **Evening:** Reading practice (N4 level)
- **Weekend:** Listening practice (slow news)

**Milestones:**
- [ ] 1,500 vocab words (N4 complete)
- [ ] 300 kanji characters
- [ ] Transitive/intransitive verb pairs
- [ ] Te-form mastery
- [ ] Potential form (〜られる)
- [ ] Causative basics (〜させる)

**Key Focus:**
- Transitive vs intransitive verbs ( crucial for N3! )
- Te-form combinations (てください・ています・てもいい)
- Conditionals (たら・ば・と)
- N4 grammar patterns (100+)

---

### Phase 3: N3 Mastery (Months 6-10)
**Goal:** Full N3 coverage - The Big Jump

**Daily Schedule:**
- **Morning:** 20 N3 vocab (new + review)
- **Midday:** 10 N3 grammar patterns
- **Evening:** Reading (NHK Easy News)
- **Weekend:** Listening (anime without subs)

**Milestones:**
- [ ] 3,700 vocab words (N3 complete)
- [ ] 650 kanji characters
- [ ] Honorifics (尊敬語・謙譲語)
- [ ] Causative-passive (〜させられる)
- [ ] Complex conditionals
- [ ] Grammar pattern nuances

**Key Focus:**
- Honorific language (keigo) - essential for work
- Grammar nuance distinctions (わけ・はず・べき)
- Reading speed improvement
- Natural conversation flow
- Business basics

**Critical Patterns:**
- 〜わけにはいかない vs 〜わけがない
- 〜はず vs 〜べき
- 〜ところ (about to/in progress/just finished)
- 〜のに (despite/contradiction)
- 〜ものだ (general truth)

## The Reality Check

**14 months is fast but doable** for a working engineer because:
1. You have programming logic (good for grammar patterns)
2. You already started (not from zero)
3. You have motivation (visa requirement)
4. You can build tools (like Kotoba) to help yourself

**Risk points:**
- Months 5-7 (N3 jump) - hardest transition
- Maintaining daily habit during work crunch
- Kanji writing practice (easy to skip)

**Success factors:**
- Daily exposure (minimum 30 min)
- Consistency beats intensity
- Active recall (Kotoba's approach)
- Finding enjoyment in the process
